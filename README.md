# 福彩3D数据分析器

自动抓取福彩3D开奖数据，进行遗漏期数分析、胆码规律分析、规则命中率统计，并生成可视化HTML报告。

## 文件说明

| 文件 | 说明 |
|---|---|
| index.html | 可视化HTML报告（主页面） |
| rule_analyzer.py | 9条胆码规律规则分析脚本 |
| fc3d_history.csv | 806期历史开奖数据（2024-2026） |
| rule_detailed_full_806.csv | 完整详细触发记录（830条） |
| fetch_3d_lottery.py | 数据抓取脚本 |
| analyze_3d_lottery.py | 基础分析脚本 |
| analyze_3d_advanced.py | 高级分析+投注+可视化脚本 |

## 在线报告

https://maxinemak1217-pixel.github.io/lottery-3d-analyzer/

## 功能模块

- **号码生成器**：生成全部1000注，支持过滤条件
- **走势分析**：任两码和差、跨度走势预测
- **预测引擎**：胆码预测 + 9条规则引擎
- **万能七码**：六组万能七码覆盖匹配
- **跨度分析**：跨度分布、走势与回调预测
- **万能组合**：6组万能两码组合检测
- **规则统计**：基于806期真实数据的9条规则命中率分析（新增）
- **3D数据看板**：遗漏期数、冷热号、投注推荐

## 规则统计功能（新增）

基于2024-2026年共806期真实开奖数据，对9条胆码规律进行命中率统计：

| 功能 | 说明 |
|---|---|
| 命中率表格 | 展示每条规则的1期/2期/3期命中率 |
| 规则评级 | S/A/B/C/D五级评级，快速识别有效规则 |
| 规则编辑器 | 修改规则条件中的关键数字，实时重新验证 |
| 详细记录 | 查看每条规则的最近10次触发记录 |
| 导出CSV | 导出统计结果和详细记录为CSV |

## 使用方法

```bash
# 运行规则分析（基于806期历史数据）
python rule_analyzer.py fc3d_history.csv

# 抓取最新数据
python fetch_3d_lottery.py

# 高级分析+可视化
python analyze_3d_advanced.py
```
